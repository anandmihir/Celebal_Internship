## Week 1 Assignment

This repository contains my Week 1 assignment completed as part of the **Celebal Technologies Data Science Internship**. The notebook covers the fundamental concepts of Python, NumPy, Pandas, Linear Algebra, Statistics, and Probability that form the foundation of Data Science and Machine Learning.

---

## Topics Covered

### Part 1 – Python Fundamentals
- Data Types & Control Flow
- Data Structures
- Exception Handling
- Functions and Lambda Expressions

### Part 2 – NumPy
- Array Creation and Reshaping
- Indexing & Slicing
- Matrix Operations
- Vectorized Computations

### Part 3 – Pandas
- DataFrames vs Series
- Data Selection using `loc` and `iloc`
- Filtering & GroupBy
- Handling Missing Data

### Part 4 – Linear Algebra
- Vector Visualization
- Matrix Operations
- Eigenvalues & Eigenvectors
- Singular Value Decomposition (SVD)
- Dimensionality Reduction

### Part 5 – Statistics
- Descriptive Statistics
- Inferential Statistics
- Hypothesis Testing
- Error Metrics (MAE, MSE, RMSE, R², Adjusted R²)
- Distribution Testing
- Stationarity (ADF Test)
- Population Stability Index (PSI)

### Part 6 – Probability Theory
- Basic Probability
- Probability Distributions
- Bayes' Theorem
- Central Limit Theorem (CLT)
